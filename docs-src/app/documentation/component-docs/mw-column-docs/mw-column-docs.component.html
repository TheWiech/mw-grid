Overview

The MwColumnDirective is used to define and customize the behaivor of a columnn in the grid. The directive must be a direct child of mw-grid.

If width, minWidth, and maxWidth are not set the column defaults to a fixed width of 150px. If the minWidth is set and no maxWidth or width is set the minWidth is used as the width. If the maxWidth is set and no minWidth or width is set the maxWidth is used as the width.

Bindings
(Required) The property on the item containing the text to display. The width of the column. The value can be expressed in pixels or a star sized value. Star sized values allow the column to dynimcally resize based on the remaining space available after non star sized columns widths have been detirmined and compared to other star sized colums. E.g. If a grid has two columns and the fist column has a width of 1* and the second column has a width of 2* the first column's width will be 33% of available space and the second column's width will be 66% of available space. The minimum width of the column in pixels. If width is set this value will have no affect. The maximum width of the column in pixels. If width is set this value will have no affect.
